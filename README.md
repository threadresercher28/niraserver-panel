# Панель управления NiraServer.

- Управления группами
- управления пользователями
- Массовое добавление.

# Управления каналами:

Редактирования:<br>
— ID<br>
— Class<br>
— Название<br>
— Ссылка на логотип

Управление каналами относящихся к определенной группе.<br>
⚠️ Нельзя создать из меню группы только из редактора каналов.
